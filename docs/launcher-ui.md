# 🖥 Launcher UI

The Launcher provides a graphical interface for running the original Xspatula setup workflow.

---

# Dashboard

The home screen contains four sections.

## Create Database

Creates

- PostgreSQL database
- Schemas
- Tables
- Roles
- Environment files

---

## Setup Processes

Registers

- Processes
- Parameters
- Dependencies

using the original setup workflow.

---

## Delete Database

Deletes the configured database.

A confirmation dialog is displayed before execution.

---

## Learning Game

The launcher embeds an interactive game.

```
https://exhibitiongames.se/game/the-risk-arena/r5/
```

The game is displayed inside an iframe.

Its purpose is to introduce users to the overall Xspatula architecture before they begin configuring a project.

---

# Progress Tracking

The launcher provides

- Progress bar
- Running status
- Success state
- Failure state

---

# Console

The console displays the original output generated by Xspatula.

Nothing is hidden.

This makes debugging significantly easier.

---

# Confirmation Dialogs

Before running any destructive action the launcher displays a confirmation dialog.

Examples

- Create Database
- Setup Processes
- Delete Database

The dialog explains exactly what will happen.

---

# Design Goals

The UI is intentionally simple.

Goals

- Fast
- Clear
- Modern
- Beginner friendly

The Launcher should feel like an installer rather than a developer tool.
